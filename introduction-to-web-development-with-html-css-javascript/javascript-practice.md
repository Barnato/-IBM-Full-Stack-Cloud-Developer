# Javascript practice

* Command + T to open new tab on Google browser.
*
* Right click and hit **Inspect**.
* Click on **Console**.
* Type

```javascript
console.log("Hello world!")
```


  ![var-log](/assets/var-log.png)

* Both `let` and `var` can be used to create variables:

```javascript
let num = 5
var mystr = "John"
console.log(num)
console.log(mystr)
```
