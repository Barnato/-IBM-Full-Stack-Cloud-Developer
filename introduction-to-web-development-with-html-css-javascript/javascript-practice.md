# Javascript practice

1. Command + T to open new tab on Google browser.
1. Right click and hit **Inspect**.
1. Click on **Console**.
1. Type:

```javascript
console.log("Hello world!")
```


![var-log](/assets/var-log.png)

* Both `let` and `var` can be used to create variables:

```javascript
let num = 5
var mystr = "John"
console.log(num)
console.log(mystr)
```
